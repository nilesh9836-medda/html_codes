# 01. HTML Fundamentals

- ## How to write a simple webpage?

```html
<!DOCTYPE html>
<html lang="en-us">
    <head>
        <meta charset="utf-8" />
        <meta name="viewport" content="width=device-width initial-scale=1.0" />
        <title>A Simple Webpage</title>
    </head>
    <body>
        <h1>Hello World</h1>
        <p>This is a simple paragraph</p>
    </body>
</html>
```

Now, save this code with `.html` extension in a folder. Then open the file with any browser. You will see your first webpage. *[Click here to see the output](./001simplewebpage.html)*

- ## How to create headings in webpage?

```html
<body>
    <h1>heading 1</h1>
    <h2>heading 2</h2>
    <h3>heading 3</h3>
    <h4>heading 4</h4>
    <h5>heading 5</h5>
    <h6>heading 6</h6>
</body>
```
